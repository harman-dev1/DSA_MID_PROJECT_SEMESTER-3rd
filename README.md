# 🛒 eBay Data Scraper & Sorting Visualizer

A Python-based desktop application that scrapes product data from eBay and allows users to visualize and interact with multiple sorting and searching algorithms in real time through a graphical interface.

---

## 🚀 Features

* 🔍 Scrapes real-time product data from eBay using **BeautifulSoup**
* 🖥️ Interactive GUI built with **PyQt5**
* 📊 Implements **8 Sorting Algorithms**:

  * Quick Sort
  * Merge Sort
  * Bubble Sort
  * Insertion Sort
  * Selection Sort
  * Bucket Sort
  * Radix Sort
  * Counting Sort
* 🔎 Includes **Linear Search** functionality
* ⚡ Dynamic, real-time display of sorted and searched data
* 📁 Handles large datasets efficiently
